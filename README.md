# 🤖 Robot Battery Monitor v2.0

Fun & powerful battery log analyzer with live mode, graphs, reports, and more!

## Quick Start
1. `pip install -r requirements.txt`  (or just `pip install matplotlib`)
2. Run: `python robot_battery_monitor.py --plot --html --csv`
3. For live monitoring: `python robot_battery_monitor.py --live`

## All Options
```bash
python robot_battery_monitor.py --help
```
